Hyprland

Hyprland is a wlroots-based tiling Wayland compositor written in C++. Noteworthy features of Hyprland include dynamic tiling, tabbed windows, a clean and readable C++ code-base, and a custom renderer that provides window animations, rounded corners, and Dual-Kawase Blur on transparent windows.

Keybindings

Here's a list of some important keybinds you need to know for operating hyprland easily. If you want to view, modify or add new keybinds, Edit ~/.config/hypr/hyprland.conf file.

  • W : Window
  • C : Control
  • S : Shift
  • A : Alt

Applications

Keybind Action
W + Return Open terminal (alacritty)
W + S + Return Open floating terminal
W + t Open fullscreen terminal
W + S + f Open file manager (thunar)
W + S + e Open text editor (geany)
W + S + w Open web browser (firefox)

Window Management

Keybind Action
W + c/q Close application
W + f Toggle fullscreen mode
W + s Toggle pseudo mode
W + Space Toggle between floating and tiled mode
W + Left/Right/Up/Down Change focus of the window directionally
W + S + Left/Right/Up/Down Swap windows directionally
W + C + Left/Right/Up/Down Resize windows directionally
W + A + Left/Right/Up/Down Move (floating) windows directionally
W + Tab Switch between windows
W + 1..8 Switch to respective workspace
W + S + 1..8 Move focused window to respective workspace
C + A + Left/Right Switch to next/previous workspace
C + A + S + Left/Right Send active window to next/previous workspace
W + C + f Toggled all floating on a workspace
W + C + s Toggled all pseudo on a workspace
W + S + p Toggled pinning
W + S + s Swap next
W + S + o Toggle opaque

Misc Keys

Keybind Action
W + p Color Picker
C + A + l Lockscreen
C + A + Delete Quit Hyprland session

Themes

pywal support has been added to hyprland. To change themes, Left Click on the theme module on waybar. To restore the default theme, Right Click on the same module on waybar.
