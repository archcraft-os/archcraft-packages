Sway

Sway is a tiling Wayland compositor and a drop-in replacement for the i3 window manager for X11. It works with your existing i3 configuration and supports most of i3's features, plus a few extras.

Keybindings

Here's a list of some important keybinds you need to know for operating sway easily. If you want to view, modify or add new keybinds, Edit ~/.config/sway/config file.

  • W : Window
  • C : Control
  • S : Shift
  • A : Alt

Applications

Keybind Action
W + Return Open terminal (alacritty)
W + S + Return Open floating terminal
W + A + Return Open fullscreen terminal
W + S + f Open file manager (thunar)
W + S + e Open text editor (geany)
W + S + w Open web browser (firefox)
C + A + v Open vim (CLI text editor)
C + A + r Open ranger (CLI file manager)
C + A + h Open htop (system monitor)

Window Management

Keybind Action
W + c/q Close application
W + h/v/g Split tiles horizontally / vertically, Toggle between h/v
W + S + s Set stacking layout
W + S + t Set tabbed layout
W + S + d Set default layout
W + S + l Toggle between stacking/tabbed/split
W + S + v Toggle between horizontal/vertical
W + f Toggle fullscreen mode
W + Space Toggle between floating and tiled mode
W + Left/Right/Up/Down Change focus of the window directionally
W + S + Left/Right/Up/Down Swap windows directionally
W + a Sets focus to the parent container
W + z Sets the focus to the last focused child container
W + Tab Automatically sets focus to the adjacent container
W + S + Space Toggles focus between floating/tiling containers
W + A + c Move floating container to the center of all outputs
W + A + p Move container to the current position of the cursor
W + A + Left/Right/Up/Down Resizing containers/windows directionally
W + o Sticky floating windows
W + y Changing border style
W + minus Make the currently focused window a scratchpad
W + S + minus Show the first scratchpad window
W + 1..8 Switch to respective workspace
W + S + 1..8 Move focused window to respective workspace
W + S + r Activate Resize Mode
W + S + m Activate Move Mode
W + S + g Activate Gaps Mode
W + S + o Activate Opacity Mode

Hardware Keys

Keybind Action

S + Print Take screenshot of desktop in 10 seconds
C + Print Take screenshot of focused window
W + Print Take screenshot of selected area

XF86AudioLowerVolume Decrease Volume

XF86AudioMicMute Toggle Mute Microphone

Themes

pywal support has been added to sway. To change themes, Left Click on the theme module on waybar. To restore the default theme, Right Click on the same module on waybar.
