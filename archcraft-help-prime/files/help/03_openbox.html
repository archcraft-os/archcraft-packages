Openbox

Openbox is a lightweight, powerful, and highly configurable stacking window manager. It allows you to change almost every aspect of how you interact with your desktop and invent completely new ways to use and control it. It's the default window manager in Archcraft.

Keybindings

Here's a list of some important keybinds you need to know for operating openbox easily. To see all the keybinds, Right click on the desktop and go to Keybinds. If you want to modify or add new keybinds, Edit ~/.config/openbox/rc.xml file.

  • W : Window
  • C : Control
  • S : Shift
  • A : Alt

Applications

Keybind Action
W + Return Open terminal (alacritty)
W + t Open fullscreen terminal
W + f Open file manager (thunar)
W + e Open text editor (geany)
W + w Open web browser (firefox)
C + A + t Open alternate terminal (xfce)
C + A + v Open vim (CLI text editor)
C + A + r Open ranger (CLI file manager)
C + A + h Open htop (system monitor)
C + A + m Open ncmpcpp (with album art)

Workspaces

Keybind Action
W + 1/2/3/4/5 Switch to respective workspace
C + A + Left/Right Switch to previous/next workspace
W + S + 1/2/3/4/5 Send active window to respective workspace
W + S + Left/Right Send active window to previous/next workspace
W + = Add new desktop
W + - Removed last desktop
W + d Toggle show desktop

Window Management

Keybind Action
W + Left/Right Move window to the left/right edge of the screen
W + Up/Down Maximize/Center window respectively
W + S + Up/Down Move window to the up/down edge of the screen
W + h/j/k/l Move window to TopLeft/BottomLeft/TopRight/BottomRight
W + S + h/j/k/l Tile window to TopLeft/BottomLeft/TopRight/BottomRight
W + A + Up/Down/Left/Right Move active window Up/Down/Left/Right relatively
W + C + Up/Down/Left/Right Resize active window relatively
W + A + m Toggle interactive move
W + A + r Toggle interactive resize
W + Tab Switch between applications (All workspaces)
A + Tab Switch between applications (Current workspaces)
W + c/q Close application
A + F4 Close application
A + F5 Iconify/Minimize current application
A + F6 Toggle maximize
A + F7 Toggle shade
A + F8 Toggle decorations
A + F9 Toggle omnipresent/sticky
A + F10 Toggle fullscreen
A + F11 Toggle always on top
A + F12 Toggle always on bottom
W + Escape Force kill application

Misc Keys

Keybind Action
W + p Colorpicker
C + A + l Lockscreen
C + S + r Reconfigure Openbox WM
C + S + BackSpace Restart Openbox WM
C + A + Delete Exit Openbox WM

Hardware Keys

Keybind Action
Print Take screenshot of desktop

S + Print Take screenshot of desktop in 10 seconds
C + Print Take screenshot of active window
W + Print Take screenshot of selected area

Styles / Themes

There are 25+ styles available for openbox, for every mood and environment, whether it's Dark or Light.
To Change the style, Right click on the desktop and go to Preferences > Change Style
